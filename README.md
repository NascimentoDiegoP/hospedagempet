Sistema criado para cadastro de informações do Donos e seus Pets, que será utilizado por proprietária de serviço de hospedagem pet.
A pagina (index.html) é a inicial onde poderar selecionar o dados cadastrais dos pets listados ao clicar sobre a opção desejada.
Com isso abrirá a pagina (prontuariopet.html) com as informações do Pet selecionado.
Para cadastrar primeiro é necessário cadastrar o Dono (cadastrodono.html), depois  poderá cadastrar o Pet (cadastropet.html) selecionando o Dono previamente cadastrado.
